# Rule-Based AI Chatbot 🤖

## Project Overview

This project was developed as part of the DecodeLabs AI Internship (Week 1).

The chatbot is a simple rule-based conversational assistant built using Python. It responds to predefined user inputs using a dictionary-based approach instead of long if-else statements.

---

## Features

- Greets users
- Responds to multiple predefined commands
- Handles unknown inputs gracefully
- Runs continuously until the user exits
- Removes punctuation from user input
- Converts input to lowercase for better matching

---

## Technologies Used

- Python 3
- Dictionary
- Loops
- String Manipulation

---

## Sample Commands

- hello
- hi
- hey
- bye
- thanks
- good morning
- good evening
- who are you
- what is python
- what is machine learning
- tell me a joke
- who created you
- exit
